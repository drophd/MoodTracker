# Веб-приложение для отслеживания настроения и самочувствия

## О проекте

Веб-приложение для мониторинга эмоционального состояния пользователя. Позволяет отмечать настроение по дням, вести заметки и анализировать динамику через графики.

<img src="https://github.com/user-attachments/assets/67e58685-f76b-4d2e-b25b-fcf4bcda0b50" alt="image" style="width: 30%; height: auto;" />



**Основные функции:**
- Календарь для выбора даты и оценки настроения
- Добавление текстовых заметок к каждому дню
- Визуализация статистики в виде графика
- Сохранение данных в формате JSON

## Технологии

- **Frontend:** HTML, CSS, JavaScript, React
- **Backend:** Python
- **Хранение данных:** JSON-файлы
- **Обмен данными:** REST API (POST-запросы)

## Как запустить проект


1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/drophd/MoodTracker.git
   
2. Запустить сервер:
    ```bash
    python app.py

3. Запустить фронтенд:
    ```bash
   npm start
    
4. Открыть в браузере: http://localhost:3000
